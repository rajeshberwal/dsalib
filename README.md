## Install

If you have installed multiple versions of python then use:

```shell
pip3 install dsalib
```

and if you have only installed Python3 then use:

```shell
pip install dsalib
```

## Usage

```python
from dsalib import Sorting
arr = [21, 2, 3, 31, 45]
Sorting.merge_sort(arr)
print(arr)
```

Output:

```shell
[2, 3, 21, 31, 45]
```
